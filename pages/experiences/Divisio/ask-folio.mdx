# Ask Folio
### Objetivo
O objetivo é ser uma plataforma que consegue localizar
 conectar clientes e experts em uma vídeo chamada rapidamente.
### Experiência
nesse projeto tive a oportunidade de começar um projeto do 
zero e desenhar soluções para problemas complexos, além de 
desenvolver uma comunicação mais limpa e assertiva com o time.
 trabalhei com os mais variados tipos de tarefas entre elas a 
 implementação de novos meios de pagamentos na plataforma como 
 pagamento via PIX, também me aprofundei em conceitos como Clean 
 Architecture , metodologia Scrum e em tecnologias como NextJs, 
 GraphQL, Prisma, postgreSQL, Git Flow, CI/CD
